# 工具说明

由于目前公开的SQLmap工具汉化版，普遍存在机器翻译、语句不通顺、拼接不符合正常中文逻辑的问题，使用过程中极度难受，因此决定人工汉化一遍，并对部分代码修改优化大概更改六千余处，纯人力修改可能存在部分未汉化的地方，如有发现问题还请联系作者反馈！
